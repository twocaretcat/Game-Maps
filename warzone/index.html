About

An interactive map of Verdansk from Warzone in Call of Duty Modern Warfare. See the locations of vehicles, ammo caches, and buy stations. Note that this map is of the original version of Verdansk, so spawn locations may no longer be accurate.

This project is licensed under the MPL 2.0 License and the source code for this project is available on GitHub. If you encounter any bugs, please open an issue.

Keep in mind that is an archived project, so I will only be able to fix issues that severely impact the functionality of the site.
